- Reference the project by name descriptively ("I use Wick's CD Tracker", "inspired by the Wick suite").

## Renaming a fork

1. Change `.toc` `## Title:` — remove "Wick's".
2. Change `## Author:` to your name.
3. Replace `logo.svg` or remove it.
4. Drop the fel-green + void palette combo, or change the accent substantially.
5. Cite the fork origin in your README (MIT attribution).
